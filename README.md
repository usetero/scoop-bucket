# Scoop Bucket for Tero

Official Scoop bucket for [Tero](https://usetero.com) - improve your observability data quality from the terminal.

## Installation

```powershell
scoop bucket add tero https://github.com/usetero/scoop-bucket
scoop install tero
```

Verify the installation:

```powershell
tero --version
```
